Project #1 Lost in Translation

Description

This is drawing I made using shape primitives with p5.

Design Process

I started with the 2D grid system, and then make the same design with p5 by using only rectangle.

Reflection

For me, the most hardest part of this coding process was put everything into right position. My design is consists of different sizes of rectangles and these shapes are all connected into edges, so it took me a so lone time to figure out right position. After writing code for this design, I learned that the hierchy is really importatnt and affects a lot on the entire design.
